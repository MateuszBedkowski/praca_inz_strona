Instructions for Linux

You need to prepare your device in order to run VulnAgent - follow below steps in order.

    Prepare system - update packages and install required modules

    Debian / Ubuntu:

    sudo apt update && sudo apt upgrade
    sudo apt install python3-tk

    Fedora:

    sudo dnf update
    sudo dnf install python3-tkinter

    Arch:

    sudo pacman -Syu
    sudo pacman -S tk

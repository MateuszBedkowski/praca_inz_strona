Instructions for Linux

In order to run VulnAgent you must have Python3 installed on your machine

Run below command to check if you have Python3 installed:

python3 --version

You should get something similar to: "Python 3.XX.X"

If you did get similar output you can download VulnAgent and execute it.

If you dit NOT get such output, please follow below steps in order to install Python3

    Debian / Ubuntu:

    sudo apt update
    sudo apt install python3

    Fedora:

    sudo dnf update
    sudo dnf install python3

    Arch:

    sudo pacman -Syu
    sudo pacman -S python3

Now you have to download VulnAgent and add permissions to allow execution and then execute it:

chmod +x VulnAgent.bin
./VulnAgent.bin

Instructions for Windows

In order to run VulnAgent you must have Python3 installed on your machine

Run below command to check if you have Python3 installed:

python --version

You should get something similar to: "Python 3.XX.X"

If you did get similar output you can download VulnAgent and execute it.

If you dit NOT get such output, please follow below steps in order to install Python3

  1. Go to https://www.python.org/downloads/
  2. Download latest Python3 and run the installer
  3. Run again the command to check if Python is installed:

    4. python --version

Now you can download and run VulnAgent as Administrator

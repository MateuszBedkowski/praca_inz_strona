Instructions for Linux

In order to run VulnAgent you must have Python3 installed on your machine

Run the below command to check if you have Python3 installed:

python3 --version

You should get something similar to: "Python 3.XX.X"

If you got similar output, you can download VulnAgent and execute it. If you did NOT get such output, follow the steps below to install Python3:

    Debian / Ubuntu:

    sudo apt update
    sudo apt install python3

    Fedora:

    sudo dnf update
    sudo dnf install python3

    Arch:

    sudo pacman -Syu
    sudo pacman -S python3

Now you have to download VulnAgent, add execution permissions, and run it:

chmod +x VulnAgent.bin
./VulnAgent.bin

Instructions for Windows

In order to run VulnAgent you must have Python3 installed on your machine

Run the below command to check if you have Python3 installed:

python --version

You should get something similar to: "Python 3.XX.X"

If you got similar output, you can download VulnAgent and execute it. If you did NOT get such output, follow the steps below to install Python3:

  1. Go to https://www.python.org/downloads/
  2. Download the latest Python3 installer and run it
  3. Run the command again to check if Python is installed:
    4. python --version
